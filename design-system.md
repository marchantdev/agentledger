# AgentLedger Design System

## House Mode: Protocol Minimal

## Color Palette
- Primary brand hue: #2563EB (Casper blue - professional, trustworthy)
- Background: #0F172A (dark navy slate)
- Surface: #1E293B (elevated card surface)
- Text primary: #F8FAFC (near-white)
- Text secondary: #94A3B8 (muted slate)
- Accent success: #10B981 (verified/confirmed)
- Accent warning: #F59E0B (pending)
- Border: #334155 (subtle separators)

## Typography
- Headlines: Inter 700 (bold, clean)
- Body: Inter 400 (readable)
- Mono: JetBrains Mono (hashes, addresses, code)
- Max 2 font families

## Spacing
- Base unit: 4px
- Content padding: 24px
- Card gap: 16px
- Section gap: 48px

## Component Styles
- Cards: rounded-lg, border border-slate-700, bg-slate-800
- Buttons: rounded-md, bg-blue-600 hover:bg-blue-500
- Input fields: bg-slate-900, border-slate-600, focus:ring-blue-500
- Timeline: vertical line with dot markers, alternating sides
- Explorer links: underline, text-blue-400

## Mood Reference
- Professional, institutional, trustworthy
- Similar to: Etherscan, Dune Analytics, Casper Explorer
- NOT: neon, gaming, meme, or generic AI purple/black
